## 修改属性默认值的特性  

1， 数据属性    

[[Configurable]]，[[Enumerable]]，[[Writable]]，[[Value]]
 
`Object.defineProperty()`
   
创建一个只读的属性   

```javascript
var person = {};
Object.defineProperty(person, "name", {
writable: false, //只读
value: "Nicholas"
});
alert(person.name); //”Nicholas”
person.name = "Greg";
alert(person.name); //”Nicholas”
```   

创建一个不可配置的属性   

```javascript
var person = {};
Object.defineProperty(person, "name", {
configurable: false,
value: "Nicholas"
});
alert(person.name); //”Nicholas”
delete person.name;
alert(person.name); //”Nicholas”
```   
configurable, enumerable（枚举） 和 writable 的属性默认是false
